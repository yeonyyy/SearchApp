# SearchApp (iOS)
* UISearchBar를 이용한 서칭 앱 구현

### 개발 환경
* Swift5, iOS13, Xcode15 이상

### Includes
* Code Base UI with SnapKit
* MVVM Pattern + RxSwift, RxCocoa
